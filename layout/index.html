History of web layouts

The history of web layouts has been one of hacking the technology to make it do something it was never meant to do.

While CSS promised the separation of content and styling, it has taken a long time to achieve true separation. And the road is paved with the wrecks of well intentioned hacks.

  1. Layout tables (html is layout)
  2. Floats & negative margins (html structure is layout)
  3. Flexbox & wrap (css layout but html structure still important)
  4. Grid (css separates layout from structure)
