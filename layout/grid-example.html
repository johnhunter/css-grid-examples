A grid example

This example shows how grid-template properties can help with responsive layouts.

We map page elements to named areas and configure a layout for specific screen size and feature support.

A 3 column layout is used if the browser supports grid and is greater than 690 width. The Nav column is fixed width with article and sidebar using fractions of the remaining space. We set the minimum hight of the article so it fill the available viewport height.
